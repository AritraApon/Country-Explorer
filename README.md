# 🌍 Country Explorer

A simple and modern web application to explore countries around the world with detailed information like name, population, region, capital, and more.

🔗 Live Site: https://aritraapon.github.io/Country-Explorer/


---

## 🚀 Features

- 🌎 Browse all countries
- 🔍 Search countries by name
- 📊 View detailed country information:
  - Name
  - Capital
  - Population
  - Region
  - Flag
- ⚡ Fast and responsive UI
- 📱 Mobile-friendly design

---

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)
- REST Countries API
